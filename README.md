# Dolar-AnkhMorpork
Kryptowaluta "dolar ankhmorporski" Projekt zakłada stworzenie opartego o sieć peer-to-peer systemu prostej kryptowaluty.  Zostanie ona celowo "osłabiona" poprzez wykorzystanie słabych kryptograficznie algorytmów, aby umożliwić prezentację działania systemu na przeciętnych komputerach przenośnych. Kryptowaluta oparta będzie o technologię blockchain.
